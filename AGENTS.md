# Residale Config (Residale)

TanStack Start + Vite app. Deployed via Coolify (Docker, Node server) from
`main`.

- `npm run dev` — start the local dev server.
- `npm run build` — production build (nitro `node-server` preset), output in
  `.output/`.
- Build artifacts, plans, and other working files belong under `_resources/`
  per this repo's conventions — not in the project root.
